## Who We Are
TRAE Co-Creation Projects is a community-driven platform that catalogs and showcases collaborative AI coding projects built with TRAE. We believe in the power of community collaboration to create amazing things.
## What We Do
We maintain a curated collection of co-creation projects to help developers:
- Discover what's possible with TRAE's collaborative features
- Find inspiration from real-world collaborative implementations
- Learn how teams work together on AI-assisted coding projects
- Showcase their collaborative work and connect with other creators
## Project Categories
- **Web Applications** - Full-stack web apps built with AI collaboration
- **Tools & Utilities** - Developer tools, CLI apps, and productivity utilities
- **AI & Machine Learning** - Projects leveraging AI/ML capabilities
- **Open Source Libraries** - Reusable libraries and frameworks
- **Learning Resources** - Tutorials, guides, and educational content
- **Other** - Anything else that doesn't fit the above categories
## How to Submit
1. Create a new [Issue](https://github.com/trae-community/trae-co-creation-projects/issues/new)
2. Fill in the project information using our template
3. Submit and wait for review. We'll respond within 3-5 business days
### Project Requirements
- Uses TRAE as a core tool for collaboration
- Demonstrates meaningful collaboration (team project, pair coding, etc.)
- Project is accessible (public repository or live demo)
- Basic documentation or README provided
