## 🎯 What is This?

**TRAE Co-Creation Projects** is a community platform that catalogs and showcases collaborative AI coding projects built with TRAE. We believe in the power of community collaboration to create amazing things.

### 💡 Why It Matters

We maintain a curated collection of co-creation projects to help developers:

- 🔍 **Discover** what's possible with TRAE's collaborative features
- 💡 **Find inspiration** from real-world collaborative implementations
- 👥 **Learn** how teams work together on AI-assisted coding projects
- 🌟 **Showcase** their collaborative work and connect with other creators

## 📁 Project Categories

We accept projects in these categories:

| Category | Description | Examples |
| :--- | :--- | :--- |
| **🌐 Web Applications** | Full-stack web apps built with AI collaboration | E-commerce sites, social platforms, dashboards |
| **🛠️ Tools & Utilities** | Developer tools, CLI apps, productivity utilities | Code generators, automation scripts, dev tools |
| **🤖 AI & Machine Learning** | Projects leveraging AI/ML capabilities | ML models, AI-powered apps, data analysis tools |
| **📦 Open Source Libraries** | Reusable libraries and frameworks | UI component libraries, utility packages, SDKs |
| **📚 Learning Resources** | Tutorials, guides, and educational content | Course projects, tutorial series, workshops |
| **🎨 Other** | Anything else that doesn't fit above | Games, mobile apps, IoT projects, art installations |

## 🚀 How to Submit Your Project

Submitting your project is easy! Just follow these steps:

### Step 1: Check Requirements ✅

Before submitting, make sure your project:
- ✅ Uses **TRAE** as a core tool for collaboration
- ✅ Demonstrates **meaningful collaboration** (team project, pair programming, etc.)
- ✅ Is **accessible** (public repository or live demo available)
- ✅ Has **basic documentation** or README

### Step 2: Create an Issue 📝

1. Click **[New Issue](https://github.com/trae-community/trae-co-creation-projects/issues/new)**
2. Select the **"Project Submission"** template
3. Fill in all the required information

### Step 3: Wait for Review ⏳

- You'll receive an acknowledgment within **24 hours**
- Our team will review within **3-5 business days**
- We may ask for clarifications if needed
- Once approved, your project will be showcased!

👉 For detailed submission guidelines, see **[CONTRIBUTING.md](./CONTRIBUTING.md)**.
