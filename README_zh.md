## 我们是谁

TRAE 共创项目清单是一个由社区驱动的平台，用于收录和展示基于 TRAE 构建的 AI 协作编程项目。我们相信社区协作的力量，能够创造出令人惊叹的作品。

## 我们做什么

我们维护着一个经过筛选的共创项目合集，帮助开发者：

- 探索 TRAE 协作功能的各种可能性
- 从真实的协作项目中获取灵感
- 学习团队如何协同进行 AI 辅助编程
- 展示自己的协作成果，连接更多创作者

## 项目分类

- **Web 应用** - 基于 AI 协作构建的全栈 Web 应用
- **工具与实用程序** - 开发者工具、CLI 应用和效率工具
- **人工智能与机器学习** - 利用 AI/ML 能力的项目
- **开源库与框架** - 可复用的库和框架
- **学习资源** - 教程、指南和教育内容
- **其他** - 不属于上述类别的任何项目

## 如何提交

1. 创建新的 [Issue](https://github.com/trae-community/trae-co-creation-projects/issues/new)
2. 使用模板填写项目信息
3. 提交并等待审核，我们会在 3-5 个工作日内回复

### 项目要求

- 以 TRAE 作为核心协作工具
- 展示有意义的协作（团队项目、结对编程等）
- 项目可访问（公开仓库或在线演示）
- 提供基础文档或 README
