## 🎯 这是什么？

**TRAE 共创项目**是一个社区平台，用于收录和展示使用 TRAE 构建的 AI 协作编程项目。我们相信社区协作的力量能够创造出令人惊叹的作品。

### 💡 为什么重要

我们维护着一个经过筛选的共创项目合集，帮助开发者：

- 🔍 **探索** TRAE 协作功能的各种可能性
- 💡 **获取灵感** 从真实的协作实现中学习
- 👥 **学习** 团队如何协同进行 AI 辅助编程
- 🌟 **展示** 自己的协作成果，连接更多创作者

## 📁 项目分类

我们接受以下类别的项目：

| 类别 | 描述 | 示例 |
| :--- | :--- | :--- |
| **🌐 Web 应用** | 使用 AI 协作构建的全栈 Web 应用 | 电商网站、社交平台、仪表盘 |
| **🛠️ 工具与实用程序** | 开发者工具、CLI 应用、效率工具 | 代码生成器、自动化脚本、开发工具 |
| **🤖 人工智能与机器学习** | 利用 AI/ML 能力的项目 | ML 模型、AI 驱动的应用、数据分析工具 |
| **📦 开源库与框架** | 可复用的库和框架 | UI 组件库、工具包、SDK |
| **📚 学习资源** | 教程、指南和教育内容 | 课程项目、教程系列、研讨会 |
| **🎨 其他** | 不属于上述类别的任何项目 | 游戏、移动应用、IoT 项目、艺术装置 |

## 🚀 如何提交你的项目

提交项目很简单！只需按照以下步骤操作：

### 步骤 1：检查要求 ✅

提交前，确保你的项目：
- ✅ 使用 **TRAE** 作为核心协作工具
- ✅ 展示**有意义的协作**（团队项目、结对编程等）
- ✅ **可访问**（有公开仓库或在线演示）
- ✅ 有**基础文档**或 README

### 步骤 2：创建 Issue 📝

1. 点击 **[New Issue](https://github.com/trae-community/trae-co-creation-projects/issues/new)**
2. 选择 **"Project Submission"** 模板
3. 填写所有必需信息

### 步骤 3：等待审核 ⏳

- 你将在 **24 小时内** 收到确认
- 我们的团队会在 **3-5 个工作日** 内审核
- 如有需要可能会要求澄清
- 审核通过后，你的项目将被展示！

👉 详细的提交指南请查看 **[CONTRIBUTING.md](./CONTRIBUTING.md)**。
